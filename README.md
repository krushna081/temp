# 🚀 Krushna Jadhav | Cybersecurity Engineer & Developer Portfolio

A premium, interactive, and high-performance developer portfolio built with **React**, **Vite**, and **Tailwind CSS**. This project showcases a blend of professional experience, cybersecurity research, and software development projects with a modern, glassmorphic aesthetic.

Live Demo: [krushna081.online](https://krushna081.online)

---

## ✨ Key Features

- **🛡️ Cybersecurity Focused**: Sections dedicated to security research, vulnerability assessment, and ethical hacking.
- **📟 Interactive Terminal**: A fully functional browser-based terminal with custom commands like `resume`, `about`, `projects`, and `clear`.
- **💎 Premium Design**:
  - **Glassmorphism**: Elegant transparent cards and overlays.
  - **Animations**: Smooth transitions using **Framer Motion**.
  - **Dynamic Background**: Floating technology logos and interactive gradient orbs.
  - **Dark/Light Mode**: Seamless theme switching with persistent preferences.
- **📄 Resume Integration**: Seamless access to the resume (`resume.pdf`) across all sections (Header, Hero, About, Contact, Terminal, and Footer).
- **📱 Fully Responsive**: Optimized for every screen size, from mobile phones to large desktop monitors.

## 🛠️ Technology Stack

| Layer | Technology |
| :--- | :--- |
| **Frontend Framework** | React 19 + Vite 6 |
| **Styling** | Tailwind CSS 4 (Vanilla CSS fallback) |
| **Animations** | Framer Motion |
| **Icons** | React Icons (FontAwesome, Feather, Hi) |
| **Interactive Logic** | react-scroll, react-type-animation, react-intersection-observer |
| **Deployment** | GitHub Pages |

## ⌨️ Terminal Commands

Experience the portfolio through the terminal! Available commands:

| Command | Description |
| :--- | :--- |
| `help` | List all available commands |
| `about` | Show a brief biography |
| `projects` | List key development and research projects |
| `resume` | View or download the resume (PDF) |
| `contact` | Show social and contact details |
| `clear` | Clear the terminal screen |

## 🚀 Getting Started

To run this project locally, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/krushna081/temp.git
   cd temp
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```
